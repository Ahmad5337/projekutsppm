Weather App adalah aplikasi cuaca sederhana berbasis **React Native** menggunakan **Expo** dan **Expo Router**.  
Aplikasi ini memungkinkan pengguna mencari cuaca berdasarkan nama kota, menampilkan suhu, deskripsi cuaca, dan detail lebih lengkap seperti kelembapan, tekanan, serta kecepatan angin.


## Fitur Utama
- Cari cuaca berdasarkan nama kota  
- Menampilkan suhu, kondisi cuaca, dan detail lengkap  
- Navigasi menggunakan **Expo Router**  
- UI sederhana dan mudah dipahami  


##  Cara Menjalankan Proyek

### 1. Clone atau download project
```sh
    npm install
    npx expo start
```
